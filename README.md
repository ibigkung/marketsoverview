# Market Overview Dashboard

Multi-Timeframe Analysis (MTA) dashboard built with Next.js 15, TypeScript, and Tailwind CSS.

## Features

- 🔥 **Multi-Timeframe Analysis** — 6 simultaneous TradingView charts (1m, 5m, 15m, 1H, 4H, 1D)
- 🎯 **Asset Filters** — Quick switching between Stock, Gold, Silver, BTC, Crypto, and Forex
- 🔍 **Symbol Search** — Search any pair with `EXCHANGE:SYMBOL` format
- 🖥️ **Expand View** — Click Expand on any chart for a full-screen analysis view
- 🌙 **Dark Theme** — Professional dark mode interface
- ⌨️ **Keyboard Shortcuts** — `Ctrl+K` to focus search, `Escape` to close

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Tech Stack

- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS v4
- **Icons:** Lucide React
- **Charts:** TradingView Widget (iframe embed)
